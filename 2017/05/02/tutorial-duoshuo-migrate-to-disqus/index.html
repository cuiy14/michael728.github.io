多说评论迁移到disqus评论的教程

多说果然“寿终正寝”了：

下面介绍一下怎么将多说评论迁移到disqus上，主要分为6步：

  1. 下载迁移的评论的php脚本，下载地址：http://urouge.github.io/assets/migrate.php
  2. 导出多说评论，导出地址：http://michael-xox.duoshuo.com/admin/tools/export/，右上角“开发者中心”菜单的左边，选择当时创建的站点，左侧导航栏的“工具”-“导出数据”，默认文件名是export.json；
  3. 将1、2步骤下载的文件放在同一文件夹下。
  4. 下载php安装包，下载地址：http://windows.php.net/download#php-7.1，将php.exe的文件夹添加到系统环境变量中；
  5. 打卡终端，执行php -f migrate.php，在同级目录生成 disqus.xml 文件;
  6. 在Disqus后台:https://michaelxiang.disqus.com/admin/discussions/import/platform/generic/,择Generic(WXR)导入disqus.xml文件。
