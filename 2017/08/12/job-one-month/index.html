入职一个月的札记

发表于 2017-08-12


图片中的是詹姆，权利的游戏中，在第一季中我就期盼赶紧去死的家伙，然而，就是这个曾经的弑君者、和姐姐乱伦的家伙，竟然慢慢让我心中钦佩起来，世人都耻笑他对疯王的背叛，都耻笑他和瑟曦的奸情，但是从第一季到第七季，见证了他的成熟，内心的骑士精神埋藏在骨子里，就像图片中一个人拿着长矛冲向火龙的勇气！小恶魔出生就是侏儒，连自己的父亲都嫌弃，而詹姆却对这个侏儒弟弟一直很照顾，兄弟之情很珍贵。詹姆最大的错就是爱上了他的姐姐，瑟曦，明知是错的事情，他也会选择负了天下人！

入职整整一个月，也无风雨也无晴，时光匆匆。

人生总是充满了未知性，16年参加校招签约之后，就觉得会定居深圳了。然而呢，东莞培训完事儿才回深圳待了一天半就有了另外一种安排，哈哈，不过，我觉得这样的安排挺好 :)

应该说，不管是怎样的缘分巧合，还是要感谢一些人，深圳原部门的唐姐、苏姐、明哥、伟哥，他们都是短短相处几天的同事，虽然很短暂，但是还是让我这个刚出校园的小菜鸟感受到了暖意。再其次，还要感谢现在部门的一些同事，亮哥、坤姐、小超姐还有其他同事，当然，枫哥作为部门老大也肯定对我能够调回杭州有着很大的帮助的！缘分让我认识了一群善良的同事，一个月的时间，我已经渐渐习惯了西安这边的工作节奏，希望自己再接再厉，尽快掌握工作的技能，哦对了，宏哥这个人也是学习的榜样！

决定无所谓真正的最好，即将远离部门大佬们独自一人在杭州和业务在一起，有利有弊，只能努力做好自己的工作，将决定慢慢变为正确！

配置管理要做好，我觉得还是要多思考，缺乏思考的工作，可能就会像个木头人一样。

立个Flag，下半年除了做好交接的工作之外，要掌握下面的技能：

  1. Java
  2. 持续集成、持续测试、持续部署框架的学习
  3. Git熟练
