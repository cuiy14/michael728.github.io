2016年5月

实习前前后后已经折腾了1个月的时间，已经拿到手的offer是大连这边的SAP。银联的offer还不知道能不能拿到，一直在等待……等待真让人煎熬！

如果能拿到银联，我想，我肯定会选择去上海的……地域，真的是很关键的因素啊！

发现了一个博客，绿萝间-译文-教程Mastering Machine Learning with scikit-learn

这周搞定它！

今天是2016-05-24，下午在体育馆打羽毛球的时候，收到了银联的实习offer邮件，好激动，漫长的等待终于结束了！

下面就是一心搞学术的时间了，抓紧时间搞，不然不好意思跟“老板”开口申请去实习啊！

我记得电话面试那会儿，问我了一些NLP方向的问题，路漫漫其修远兮啊！很多东西等待区学！Fighting!

不知不觉，离去实习就剩下一周多的时间了，心里相当的忐忑啊，并不是对实习忐忑，而是对毕业，学校要求发期刊论文啊……目前的进度让我一直焦虑不安啊！！！虽然拿到的实习offer在别人看来还不错，但是，只有我自己知道心里的痛苦。三个月的实习时间可不短啊，论文、论文、论文……

师兄他们最近毕业各种聚餐，校园里也是毕业季的氛围，夏天骄阳似火，气温一天比一天高，心里也是各种躁动不安，需要静心！

希望能没有杂事烦扰，加油！

今天发现了一个博客sklearn-yan，博主这篇文章让我发现了他的blog，然后看到他这几年的积累，论文答辩，结婚，生子，工作等等，生活如是，他也有一个很好的习惯，每晚都会坚持读书，向他学习~

昨天中午吃午饭的时候，师娘跟我们提起了导师当年的生活。导师家里条件不好，当年本科也是贷款完成学业的，在学校读硕士期间，学费不够，通过同学帮助最后花了两年时间就硕士毕业了！两年期间，论文发表都完成了，这是多么优秀啊！少读一年，就少交一年的学费啊……师娘说，在日本读博期间，老师有时候点外卖丰富，一份外卖的菜分成两份，中午吃一份，另外一份当他的晚饭。

每个现在貌似成功的人，他们当年都付出了我们不知道的汗水，他们的曾经的艰辛也是现在的我们无法体会到的，值得学习的品质！所以，我要记录下这些！

不能铺张浪费，需要的东西，即使贵点也没关系，买就买个好，但是，不是必须要用的，就要懂得节约！所以，我的这款用家教钱攒下来的苹果6就让它一直为我服役下去吧！为自己而生活，不要在意那些虚荣的东西！！！
