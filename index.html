Python正则表达式学习摘要及资料

发表于 2016-07-28 | 分类于 正则 | | 阅读次数

摘要

  • 在正则表达式中，如果直接给出字符，就是精确匹配。
  • {m,n}? 对于前一个字符重复 m到 n 次，并且取尽可能少的情况 在字符串'aaaaaa'中，a{2,4} 会匹配 4 个 a，但 a{2,4}? 只匹配 2 个 a。
    ^表示行的开头，^\d表示必须以数字开头。

$表示行的结束，\d$表示必须以数字结束。

你可能注意到了，py也可以匹配'python'–>py;
但是加上^py$就变成了整行匹配，就只能匹配’py’了,匹配'python'时，就什么也得不到。
